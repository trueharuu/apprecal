# Topic: Operations on Functions

| Name       | Input              | Output              |
| ---------- | ------------------ | ------------------- |
| Sum        | $(f + g)(x)$       | $f(x) + g(x)$       |
| Difference | $(f - g)(x)$       | $f(x) - g(x)$       |
| Product    | $(fg)(x)$          | $f(x)g(x)$          |
| Quotient   | $(\frac{f}{g})(x)$ | $\frac{f(x)}{g(x)}$ |
| Composite  | $(f \circ g)(x)$   | $f(g(x))$           |

1. Given `f(x) = x^2 - 3x + 2` `g(x) = 2x - 4`
- `(f + g)(x)`
> ```rs
> = f(x) + g(x)
> = x^2 - 3x + 2 + 2x - 4
> = x^2 - x - 2
> ```
- `(fg)(x)`
> ```rs
> = f(x) * g(x)
> = (x^2 - 3x + 2) * (2x - 4)
> = (2x^3 - 6x^2 + 4x) + (-4x^2 + 12x - 8)
> = 2x^3 - 10x^2 + 16x - 8
> ```
- `(f / g)(x)`
> ```rs
> = f(x) / g(x)
> = (x^2 - 3x + 2) / (2x - 4)
> = (x^2 - 3x + 2) / 2(x - 2)
> = (x - 1)(x - 2) / 2(x - 2)
> = (x - 1) / 2
> ```
- `f(g(x))`
> ```rs
> = g(x)^2 - (3 * g(x)) + 2
> = (2x - 4)^2 - 3(2x - 4) + 2
> = (2x - 4)(2x - 4) - 6x + 12 + 2
> = 4x^2 - 16x + 16 - 6x + 14
> = 4x^2 - 22x + 30
> = 2(2x^2 - 11x + 15)
> // Factors of 30 (2 * 15) that add to -11 are { -5, -6 }
> = 2(2x^2 - 5x - 6x + 15)
> = 2(x(2x - 5) - 3(2x - 5))
> = 2(x - 3)(2x - 5)
> ```

2. Given `f(x) = 4 / x` `g(x) = 2x / (x - 2)`
- `(f - g)(x)`
> ```rs
> = f(x) - g(x)
> = (4 / x) - (2x / (x - 2))
> = (4(x - 2) / x(x - 2)) - (2x^2 / x(x - 2))
> = (4(x - 2) - 2x^2) / x(x - 2)
> = (4x - 8 - 2x^2) / x(x - 2)
> = -2(x^2 - 2x + 4) / x(x - 2)
> ```

- `(fg)(1)`
> ```rs
> = f(1)g(1)
> = (4 / 1)(2(1) / (1 - 2))
> = 4(-2)
> = -8
> ```

- `f(g(x))`
> ```rs
> = 4 / g(x)
> = 4 / (2x / (x - 2))
> = 4(x - 2) / 2x
> = 2(x - 2) / x
> ```

- `g(f(x))`
> ```rs
> = 2(f(x)) / (f(x) - 2)
> = 2(4 / x) / ((4 / x) - 2)
> = 8 / x((4 / x) - 2)
> = 8 / (4 - 2x)
> = 8 / 2(2 - x)
> = 4 / (2 - x)
> ```
