# Topic: Factoring Review

## Multiply the following polynomials.

1. `(x + 5)(2x - 3)`
> ```rs
> = 2x² + 7x - 15
> ``` 
2. `(2x - 5)²`
> ```rs
> = (2x - 5)(2x - 5)
> = 4x² - 20x + 25
> ```
3. `(x² - 2x + 1)(x + 3)`
> ```rs
> = x³ - 2x² + x + 3x² - 6x + 3
> = x³ + x² - 5x + 3
> ```

## Factor each polynomial completely.

4. `x² - 6x + 8`
> ```rs
> = (x - 4)(x - 2)
> ```
5. `27x² - 48`
> ```rs
> = 3(9x² - 16)
> = 3(3x - 4)(3x + 4)
> ```
6. `-15x² - 5x`
> ```rs
> = -5x(3x + 1)
> ```
7. `4x² + 12x + 9`
> ```rs
> = (2x + 3)²
> ```
8. `2x² - 3x - 9`
> ```rs
> = (2x + 3)(x - 3)
> ```
9. `x³ - 4x² + 2x - 8`
> ```rs
> = (x³ - 4x²) + (2x - 8)
> = x²(x - 4) + 2(x - 4)
> = (x - 4)(x² + 2)
> ```
10. `8x³ - 10x² - 3x`
> ```rs
> = x(8x² - 10x - 3)
> = x(8x² - 12x + 2x - 3)
> = x(2x(4x + 1) - 3(4x + 1))
> = x(2x - 3)(4x + 1)
> ```
11. `x³ - 8`
> ```rs
> // Difference of cubes: a³ - b³ = (a - b)(a² + ab + b²)
> = (x - 2)(x² + 2x + 4)
> ```
12. `x⁴ - 81`
> ```rs
> // Difference of squares: a² - b² = (a + b)(a - b)
> = (x² + 9)(x² - 9)
> = (x² + 9)(x + 3)(x - 3)
> ```
